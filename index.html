Irqbalance

Welcome to the new home of irqbalance. Irqbalance is a daemon that monitors the cpu load created by various interrupt sources and attempts to distribute that load over the available cpus in your system in an attempt to better balance system latency and throughput.

Development

You are welcome to use the existing infrastructure in github to create pull requests and the like, but the project maintainers are old and like doing things in the old ways. As such, development discussions take place on the irqbalance mailing list, which can be subscribed to at http://lists.infradead.org/mailman/listinfo/irqbalance
